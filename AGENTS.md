# Project Overview
## Description
Клон популярной игры «Динозаврик из Хрома» (Chrome Dinosaur Game). Игрок управляет динозавриком, который должен прыгать через кактусы, чтобы набрать максимальный счет.

## Structure
- `assets/`: Static assets (images, etc.)
- `js/`: Client-side scripts
- `index.html`: Entry point
- `style.css`: Global styles

## Technologies
- HTML5
- CSS3
- JavaScript (Vanilla)

## Architecture
- Client-side rendering
- Simple monolithic structure
