# Dino Runner

A faithful clone of the popular "Chrome Dinosaur Game". Play as a dinosaur jumping over cacti to achieve the highest score possible.

## 🚀 Live Game
You can play the game online here: [https://dino.bogatyrev.uk](https://dino.bogatyrev.uk)

## 📝 Project Description
This project is a web-based implementation of the Chrome Dinosaur game using modern web technologies. It features:
- Responsive canvas rendering.
- Smooth animations for the dinosaur, clouds, and obstacles.
- Dynamic difficulty scaling (increasing game speed over time).
- Score tracking system.
- Input handling for jumping.

## 🛠 Technologies
- **HTML5**
- **CSS3**
- **JavaScript (Vanilla)**

## 🎮 How to Run
To run this project locally:
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. Use the **Space** bar to start and jump.
